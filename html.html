Predictive Sales Model

This project involved building a machine learning model to predict sales trends for a retail company. I used Python with scikit-learn to preprocess the data, perform feature engineering, and train a Random Forest model. The dataset included sales data over three years, with features like product category, region, and seasonality.

After hyperparameter tuning using GridSearchCV, the model achieved a 92% accuracy on the test set. I also created visualizations with Matplotlib to present the predictions to stakeholders, helping them make data-driven decisions for inventory management.

Deep Learning Image Classifier

This project focused on building a convolutional neural network (CNN) for image classification using TensorFlow. The goal was to classify images into 10 categories, using a dataset of 50,000 images.

I designed a CNN architecture with 5 convolutional layers, followed by batch normalization and dropout to prevent overfitting. After training for 50 epochs, the model achieved a 95% accuracy on the test set. I also implemented a web interface using Flask to allow users to upload images and get real-time predictions.

Sentiment Analysis Tool

I created a sentiment analysis tool to analyze social media posts and classify them as positive, negative, or neutral. The project used Python with NLTK for natural language processing and a dataset of 10,000 posts.

I preprocessed the text by tokenizing, removing stop words, and applying stemming. Then, I trained a logistic regression model, achieving an 88% accuracy. The tool includes a simple UI built with Tkinter, allowing users to input text and see the sentiment analysis results.
